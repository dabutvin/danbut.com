A GitHub app designed to optimize your images. As you commit images to your repo ImgBot will open PRs after applying a lossless compression algorithm. Written in C# as an Azure Function, ImgBot listens to webhooks on projects with the app installed.

Images optimization can often provide some of the largest performance improvements for your website or app and ImgBot offers one of the most turn-key solutions. For projects already hosted on GitHub, image optimizations are literally a few clicks away.

ImgBot is built on top of Magick.NET, the .NET library for ImageMagick.

Venmorld and Gitfixed

These web apps are clones of each other and focused on the micro-niche of disposable analytics. Venmorld, is a real-time data visualization of payments pulled from the public Venmo feed. Gitfixed, has the same UI except it is based on git commits from the public GitHub feed that contain the word "fixed" in the message. While Venmorld charts the emoji frequency in Venmo payment messages, Gitfixed charts the frequency of common bugfix commit messages like 'format' or 'logic'. Both of these apps are built with Node.js and Highcharts and only store data for as long as your browser is open.

It's a snapshot of what's happening now to draw insights from real time current data. And it's fun :)
