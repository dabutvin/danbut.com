# danbut.com

Continuously deploying to http://danbut.com from the master branch

## Local Development

### Prerequisites

- Node.js

### Setup

```bash
npm install
```

### Running Locally

Generate the site and start the dev server with hot reload:

```bash
npm run gen
npm run serve
```

This will open the site in your browser and automatically reload when you make changes to files in `src/`.
